# JuzMate - Aplikasi Pendamping Hafalan Al-Qur'an

[![Platform](https://img.shields.io/badge/Android-7.0+-green.svg)]()
[![Kotlin](https://img.shields.io/badge/Kotlin-1.9.0-purple.svg)]()
[![License](https://img.shields.io/badge/License-Proprietary-red.svg)]()

JuzMate adalah aplikasi Android yang membantu Anda menghafal Al-Qur'an dengan metode progresif per ayat. Dilengkapi dengan audio murottal, kuis, dan pelacakan progres hafalan.

## ✨ Fitur Utama

| Fitur | Deskripsi |
|-------|-----------|
| 🎯 **Hafalan Progresif** | Fokus hafalan per ayat dengan metode terstruktur |
| 🎧 **Audio Per Ayat** | Murottal oleh Sheikh as Sudais |
| 📖 **Terjemah & Tafsir** | Terjemahan Kemenag RI |
| ✅ **Pelacakan Progres** | Tandai ayat yang sudah dihafal |
| 📝 **Kuis Hafalan** | Uji kemampuan hafalan Anda |
| 💾 **Simpan Ayat** | Bookmark ayat favorit |

## 🚀 Teknologi

- **Kotlin** - Bahasa pemrograman
- **Jetpack Compose** - UI Toolkit modern
- **Media3 ExoPlayer** - Pemutar audio
- **Coroutines** - Asynchronous programming
- **Material Design 3** - Desain antarmuka

## 📱 Minimum Requirements

- Android 7.0 (API level 24) ke atas
