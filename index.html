Kebijakan Privasi Juzmate

📅 Berlaku sejak: 22 Maret 2026

Tentang Aplikasi

Juzmate adalah aplikasi pendamping hafalan Al-Qur'an yang dikembangkan oleh JuzmateDev. Aplikasi ini membantu pengguna menghafal Al-Qur'an dengan metode progresif per ayat, dilengkapi audio murottal, kuis, dan pelacakan progres hafalan. Kebijakan privasi ini menjelaskan bagaimana kami menangani informasi Anda.

Pengumpulan Informasi

Juzmate TIDAK mengumpulkan informasi pribadi apapun dari penggunanya.

Aplikasi ini beroperasi sepenuhnya di perangkat Anda dan tidak:

  • Mengumpulkan alamat IP
  • Melacak perilaku pengguna
  • Menggunakan layanan analytics
  • Membagikan data ke pihak ketiga
  • Membutuhkan akun pengguna

Izin yang Digunakan

  • Internet - Untuk streaming audio murottal Al-Qur'an
  • Storage - Untuk menyimpan cache audio agar dapat diputar offline

Penyimpanan Data

Semua data disimpan secara lokal di perangkat Anda. Tidak ada data yang dikirim ke server eksternal manapun.

Layanan Pihak Ketiga

Aplikasi ini menggunakan library berikut untuk fungsionalitas:

  • ExoPlayer (Media3) - Pemutar audio
  • OkHttp - Request jaringan
  • Gson - Parsing JSON

Tidak ada satupun library di atas yang mengirimkan data pribadi ke server mereka.

Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui sewaktu-waktu. Perubahan akan diumumkan melalui halaman ini dengan tanggal efektif yang baru.
